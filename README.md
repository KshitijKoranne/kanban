# ◈ KanFlow

A bold, colorful Kanban board for dev task tracking. Built with Next.js, Turso, and Tailwind CSS.

## Features

- **Drag & Drop** — Reorder cards and columns with smooth interactions
- **Multiple Boards** — Organize by project with customizable backgrounds
- **Labels & Priority** — 10 dev-focused labels + 5 priority levels
- **Due Dates** — Track deadlines with overdue highlighting
- **Checklists** — Subtasks with progress tracking
- **Card Covers** — Color-code cards for quick visual scanning
- **Activity Log** — Track all board changes
- **Light / Dark Mode** — WCAG AA compliant, respects system preference
- **Fully Responsive** — Mobile, tablet, and desktop
